# Backend
***
Микросервисы приложения
# 1. Streaming Service
***
1. Загрузка песен:  
`/api/v1/uploads/uploadSong`  
2. Получить песню по id:  
`/api/v1/songs/{id}`  
3. Получить все песни:  
`/api/v1/songs`  
4. Получить альбом по id:  
`/api/v1/albums/{id}`  
5. Получить обложку альбома:  
`/api/v1/albums/{id}/artwork`  
6. Получить все альбомы:  
`/api/v1/albums`  
7. Получить исполнителя по id:  
`/api/v1/artists/{id}`  
8. Получить список альбомов исполнителя по id:  
`/api/v1/artists/{id}/albums`  
9. Получить всех исполнителей:  
`/api/v1/artists`  
10. Получить жанр по id:  
`/api/v1/genres/{id}`  
11. Получить все жанры:  
`/api/v1/genres`  
12. Получить стрим аудио по id:  
`/api/v1/streaming/stream/{id}`
